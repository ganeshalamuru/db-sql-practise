# Q048 - OUTER JOIN

- **Difficulty:** 4 / 5
- **Learning objective:** Place a related-table filter in the join condition to preserve unmatched rows.

## Task

Return supplier_id, supplier_name, and active_product_count for every supplier, including suppliers with no active products.

## Result requirements

- **Ordering:** Order by active_product_count descending, then supplier_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/10-outer-join/q048.sql`. Do not modify the schema or data.
