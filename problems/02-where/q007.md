# Q007 - WHERE

- **Difficulty:** 1 / 5
- **Learning objective:** Filter numeric values with a comparison predicate.

## Task

Return product_id, sku, name, and unit_price for active products priced above 1,000.00.

## Result requirements

- **Ordering:** Order by unit_price ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/02-where/q007.sql`. Do not modify the schema or data.
