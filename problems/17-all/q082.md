# Q082 - ALL

- **Difficulty:** 3 / 5
- **Learning objective:** Use = ALL to select rows whose related values all meet a condition.

## Task

Return order_id, ordered_at, and status for orders whose every order item has a quantity of at least 2. Exclude orders with no order items.

## Result requirements

- **Ordering:** Order by ordered_at descending, then order_id descending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/17-all/q082.sql`. Do not modify the schema or data.
