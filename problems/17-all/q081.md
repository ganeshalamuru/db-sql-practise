# Q081 - ALL

- **Difficulty:** 3 / 5
- **Learning objective:** Compare a value to every row returned by a subquery.

## Task

Return product_id, sku, name, and unit_price for active products priced higher than every active product supplied by supplier_id 1.

## Result requirements

- **Ordering:** Order by unit_price ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/17-all/q081.sql`. Do not modify the schema or data.
