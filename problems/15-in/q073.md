# Q073 - IN

- **Difficulty:** 4 / 5
- **Learning objective:** Use an aggregate subquery as the IN result set.

## Task

Return warehouse_id, warehouse_code, product_id, sku, and quantity_on_hand for inventory rows whose product_id appears in at least one return with status requested or approved.

## Result requirements

- **Ordering:** Order by warehouse_id ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/15-in/q073.sql`. Do not modify the schema or data.
