# Q055 - SELF JOIN

- **Difficulty:** 5 / 5
- **Learning objective:** Chain self joins to navigate more than one hierarchy level.

## Task

Return employee_id, employee_first_name, employee_last_name, manager_first_name, manager_last_name, grandmanager_first_name, and grandmanager_last_name for employees whose manager also has a manager.

## Result requirements

- **Ordering:** Order by employee_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/11-self-join/q055.sql`. Do not modify the schema or data.
