# Q051 - SELF JOIN

- **Difficulty:** 3 / 5
- **Learning objective:** Join an employee to its manager in the same table.

## Task

Return employee_id, employee_first_name, employee_last_name, manager_id, manager_first_name, and manager_last_name for employees who have a manager.

## Result requirements

- **Ordering:** Order by manager_id ascending, then employee_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/11-self-join/q051.sql`. Do not modify the schema or data.
