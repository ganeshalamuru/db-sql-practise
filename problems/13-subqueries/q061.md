# Q061 - SUBQUERIES

- **Difficulty:** 3 / 5
- **Learning objective:** Compare each row to a scalar aggregate returned by a subquery.

## Task

Return product_id, sku, name, and unit_price for active products priced above the average unit_price of all active products.

## Result requirements

- **Ordering:** Order by unit_price descending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/13-subqueries/q061.sql`. Do not modify the schema or data.
