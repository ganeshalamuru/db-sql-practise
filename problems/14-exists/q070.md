# Q070 - EXISTS

- **Difficulty:** 5 / 5
- **Learning objective:** Use a correlated NOT EXISTS test with multiple prior concepts.

## Task

Return warehouse_id, warehouse_code, product_id, sku, quantity_on_hand, and reorder_point for inventory rows that are at or below their reorder point and have no matching store_inventory row with a positive quantity_on_hand for the same product.

## Result requirements

- **Ordering:** Order by warehouse_id ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/14-exists/q070.sql`. Do not modify the schema or data.
