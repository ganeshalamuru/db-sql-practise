- **Difficulty:** 3 / 5
- **Learning objective:** Use NOT EXISTS to find rows without a related match.

## Task

Return product_id, sku, and name for active products that have never received a review.

## Result requirements

- **Ordering:** Order by product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/14-exists/q067.sql`. Do not modify the schema or data.
