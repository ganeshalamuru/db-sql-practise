# Q069 - EXISTS

- **Difficulty:** 5 / 5
- **Learning objective:** Combine joins and an existence test to filter a transaction result.

## Task

Return order_id, ordered_at, customer_id, customer_first_name, customer_last_name, and status for orders that contain at least one product with a unit_price of 1,000.00 or more. Include each qualifying order once.

## Result requirements

- **Ordering:** Order by ordered_at descending, then order_id descending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/14-exists/q069.sql`. Do not modify the schema or data.
