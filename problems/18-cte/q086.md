# Q086 - CTE

- **Difficulty:** 3 / 5
- **Learning objective:** Use a CTE to name an aggregate value for a comparison.

## Task

Return product_id, sku, name, and unit_price for active products priced above the average unit_price of all active products.

## Result requirements

- **Ordering:** Order by unit_price descending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/18-cte/q086.sql`. Do not modify the schema or data.
