# Q088 - CTE

- **Difficulty:** 4 / 5
- **Learning objective:** Use a CTE to isolate a filtered data set before joining it to reference tables.

## Task

Return warehouse_id, warehouse_code, product_id, sku, quantity_on_hand, and reorder_point for inventory rows at or below their reorder point, but only for active products.

## Result requirements

- **Ordering:** Order by warehouse_id ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/18-cte/q088.sql`. Do not modify the schema or data.
