# Q016 - Expressions

- **Difficulty:** 1 / 5
- **Learning objective:** Create a numeric expression and name its result.

## Task

Return order_id, product_id, quantity, unit_price, and line_total (quantity multiplied by unit_price) for every order item.

## Result requirements

- **Ordering:** Order by order_id ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/04-expressions/q016.sql`. Do not modify the schema or data.
