# Q019 - Expressions

- **Difficulty:** 2 / 5
- **Learning objective:** Create a boolean expression in the SELECT list.

## Task

Return product_id, name, unit_price, and a boolean is_premium that is true when unit_price is at least 1,000.00.

## Result requirements

- **Ordering:** Order by product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/04-expressions/q019.sql`. Do not modify the schema or data.
