# Q017 - Expressions

- **Difficulty:** 1 / 5
- **Learning objective:** Use arithmetic to calculate a difference.

## Task

Return warehouse_id, product_id, quantity_on_hand, reorder_point, and units_above_reorder (quantity_on_hand minus reorder_point) for all inventory rows.

## Result requirements

- **Ordering:** Order by warehouse_id ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/04-expressions/q017.sql`. Do not modify the schema or data.
