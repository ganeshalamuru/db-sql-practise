# Q018 - Expressions

- **Difficulty:** 2 / 5
- **Learning objective:** Combine multiple columns into a derived total.

## Task

Return order_id, shipping_fee, and shipping_fee plus 18 percent tax as shipping_fee_with_tax for every order.

## Result requirements

- **Ordering:** Order by order_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/04-expressions/q018.sql`. Do not modify the schema or data.
