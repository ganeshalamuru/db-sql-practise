# Q028 - Aggregate Functions

- **Difficulty:** 3 / 5
- **Learning objective:** Aggregate multiple columns and derive a value from aggregate results.

## Task

Return total_units_on_hand, total_reorder_point, and units_above_reorder across all inventory rows. units_above_reorder is the sum of quantity_on_hand minus reorder_point only for rows where quantity_on_hand is greater than reorder_point; do not let below-reorder rows reduce this value.

## Result requirements

- **Ordering:** The result has one row; no ordering is needed.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/06-aggregate-functions/q028.sql`. Do not modify the schema or data.
