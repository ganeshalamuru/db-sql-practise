# Q027 - Aggregate Functions

- **Difficulty:** 3 / 5
- **Learning objective:** Combine COUNT, SUM, and AVG after filtering the input rows.

## Task

For completed payments only, return completed_payment_count, completed_payment_total, and average_completed_payment_amount. Round the average amount to two decimal places.

## Result requirements

- **Ordering:** The result has one row; no ordering is needed.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/06-aggregate-functions/q027.sql`. Do not modify the schema or data.
