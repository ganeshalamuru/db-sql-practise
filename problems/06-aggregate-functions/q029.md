# Q029 - Aggregate Functions

- **Difficulty:** 4 / 5
- **Learning objective:** Produce a compact statistical summary from filtered rows.

## Task

For active products only, return active_product_count, lowest_active_product_price, highest_active_product_price, and average_active_product_price. Round the average price to two decimal places.

## Result requirements

- **Ordering:** The result has one row; no ordering is needed.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/06-aggregate-functions/q029.sql`. Do not modify the schema or data.
