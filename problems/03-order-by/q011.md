# Q011 - ORDER BY

- **Difficulty:** 1 / 5
- **Learning objective:** Sort a result by one column in descending order.

## Task

Return product_id, name, and unit_price for active products.

## Result requirements

- **Ordering:** Order by unit_price descending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/03-order-by/q011.sql`. Do not modify the schema or data.
