# Q015 - ORDER BY

- **Difficulty:** 3 / 5
- **Learning objective:** Control the placement of NULL values while sorting.

## Task

Return employee_id, first_name, last_name, and manager_id for all employees.

## Result requirements

- **Ordering:** Order by manager_id ascending with NULL manager_id values last, then employee_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/03-order-by/q015.sql`. Do not modify the schema or data.
