# Q001 - SELECT

- **Difficulty:** 1 / 5
- **Learning objective:** Select named columns from a relation.

## Task

Return each active product's product_id, sku, name, and unit_price.

## Result requirements

- **Ordering:** Order by product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/01-select/q001.sql`. Do not modify the schema or data.
