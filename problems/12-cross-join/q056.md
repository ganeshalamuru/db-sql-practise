# Q056 - CROSS JOIN

- **Difficulty:** 3 / 5
- **Learning objective:** Produce every possible pair from two reference tables.

## Task

Return country_code, country_name, warehouse_code, and warehouse_name for every possible country-and-warehouse combination.

## Result requirements

- **Ordering:** Order by country_code ascending, then warehouse_code ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/12-cross-join/q056.sql`. Do not modify the schema or data.
