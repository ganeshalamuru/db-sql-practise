# Q060 - CROSS JOIN

- **Difficulty:** 5 / 5
- **Learning objective:** Cross join a table with a derived value set to generate a complete reference matrix.

## Task

Return country_code, country_name, and payment_method for every country-and-payment-method combination. The payment methods are card, wallet, bank_transfer, and cash_on_delivery.

## Result requirements

- **Ordering:** Order by country_code ascending, then payment_method ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/12-cross-join/q060.sql`. Do not modify the schema or data.
