# Q093 - RECURSIVE CTE

- **Difficulty:** 4 / 5
- **Learning objective:** Build an employee hierarchy path while traversing a self-referencing table.

## Task

Return employee_id, first_name, last_name, manager_id, hierarchy_level, and reporting_path for every employee. reporting_path contains employee IDs from the hierarchy root to the employee, separated by ' > '.

## Result requirements

- **Ordering:** Order by reporting_path ascending, then employee_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/19-recursive-cte/q093.sql`. Do not modify the schema or data.
