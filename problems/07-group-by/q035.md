# Q035 - GROUP BY

- **Difficulty:** 4 / 5
- **Learning objective:** Filter rows before grouping and calculate multiple aggregates for each group.

## Task

For active products only, return supplier_id, active_product_count, and average_active_product_price. Round average_active_product_price to two decimal places.

## Result requirements

- **Ordering:** Order by average_active_product_price descending, then active_product_count descending, then supplier_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/07-group-by/q035.sql`. Do not modify the schema or data.
