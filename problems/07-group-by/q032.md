# Q032 - GROUP BY

- **Difficulty:** 3 / 5
- **Learning objective:** Group rows by a foreign-key value while summarizing a date range.

## Task

Return customer_id, order_count, first_ordered_at, and last_ordered_at for each customer who has placed at least one order.

## Result requirements

- **Ordering:** Order by order_count descending, then customer_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/07-group-by/q032.sql`. Do not modify the schema or data.
