# Q034 - GROUP BY

- **Difficulty:** 4 / 5
- **Learning objective:** Combine COUNT and a rounded AVG, then sort by aggregate results.

## Task

Return product_id, review_count, and average_rating for each reviewed product. average_rating must be rounded to two decimal places.

## Result requirements

- **Ordering:** Order by average_rating descending, then review_count descending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/07-group-by/q034.sql`. Do not modify the schema or data.
