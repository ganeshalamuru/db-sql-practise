# Q031 - GROUP BY

- **Difficulty:** 2 / 5
- **Learning objective:** Calculate multiple aggregates for each categorical group.

## Task

Return status, order_count, and total_shipping_fee: the number of orders and the sum of shipping_fee for each order status.

## Result requirements

- **Ordering:** Order by total_shipping_fee descending, then status ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/07-group-by/q031.sql`. Do not modify the schema or data.
