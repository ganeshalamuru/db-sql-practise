# Q076 - ANY

- **Difficulty:** 3 / 5
- **Learning objective:** Compare a value to any row returned by a subquery.

## Task

Return product_id, sku, name, and unit_price for active products priced higher than at least one active product supplied by supplier_id 1.

## Result requirements

- **Ordering:** Order by unit_price ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/16-any/q076.sql`. Do not modify the schema or data.
