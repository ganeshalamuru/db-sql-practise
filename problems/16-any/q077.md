# Q077 - ANY

- **Difficulty:** 3 / 5
- **Learning objective:** Use = ANY as an alternative membership comparison.

## Task

## Result requirements

- **Ordering:** Order by payment_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/16-any/q077.sql`. Do not modify the schema or data.
