# Q079 - ANY

- **Difficulty:** 5 / 5
- **Learning objective:** Combine ANY with joins and correlated aggregate logic.

## Task

Return warehouse_id, warehouse_code, product_id, sku, and quantity_on_hand for inventory rows whose quantity_on_hand is greater than the reorder_point of at least one other inventory row in the same warehouse.

## Result requirements

- **Ordering:** Order by warehouse_id ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `answers/16-any/q079.sql`. Do not modify the schema or data.
