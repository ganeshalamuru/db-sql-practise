# Q130 - EXCEPT

- **Difficulty:** 5 / 5
- **Learning objective:** Subtract completed-payment orders from the full order set before calculating transaction totals.

## Task

Return order_id, customer_id, ordered_at, order_status, payment_status, and item_total for orders whose payment status is not completed. item_total is the sum of quantity multiplied by unit_price across each order's line items.

## Result requirements

- **Ordering:** Order by ordered_at ascending, then order_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/26-except/q130.sql`. Do not modify the schema or data.
