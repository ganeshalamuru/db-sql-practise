# Q003 - SELECT

- **Difficulty:** 2 / 5
- **Learning objective:** Alias projected columns for a clear result contract.

## Task

Return each warehouse's code as warehouse_code, name as warehouse_name, and capacity_units as capacity.

## Result requirements

- **Ordering:** Order by warehouse_code ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/01-select/q003.sql`. Do not modify the schema or data.
