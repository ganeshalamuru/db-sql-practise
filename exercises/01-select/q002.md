# Q002 - SELECT

- **Difficulty:** 1 / 5
- **Learning objective:** Use DISTINCT to remove repeated values.

## Task

Return the distinct payment methods that have been used by completed payments.

## Result requirements

- **Ordering:** Order alphabetically by payment_method.
- **Duplicates:** Duplicate rows are not significant.

Write one PostgreSQL `SELECT` statement in `exercises/01-select/q002.sql`. Do not modify the schema or data.
