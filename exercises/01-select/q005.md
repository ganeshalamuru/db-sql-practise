# Q005 - SELECT

- **Difficulty:** 2 / 5
- **Learning objective:** Select a complete row shape only when it is the required contract.

## Task

Return all columns for countries in the reference data.

## Result requirements

- **Ordering:** Order by country_code ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/01-select/q005.sql`. Do not modify the schema or data.
