# Q004 - SELECT

- **Difficulty:** 2 / 5
- **Learning objective:** Project derived values with arithmetic expressions.

## Task

For every order item, return order_id, product_id, quantity, unit_price, and the line total (quantity multiplied by unit_price) as line_total.

## Result requirements

- **Ordering:** Order by order_id, then product_id.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/01-select/q004.sql`. Do not modify the schema or data.
