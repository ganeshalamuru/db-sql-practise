# Q054 - SELF JOIN

- **Difficulty:** 5 / 5
- **Learning objective:** Count child rows per parent while retaining rows with no children.

## Task

Return employee_id, first_name, last_name, and direct_report_count for every employee, including employees with no direct reports.

## Result requirements

- **Ordering:** Order by direct_report_count descending, then employee_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/11-self-join/q054.sql`. Do not modify the schema or data.
