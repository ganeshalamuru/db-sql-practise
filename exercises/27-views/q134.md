# Q134 - VIEWS

- **Difficulty:** 4 / 5
- **Learning objective:** Build a multi-table operational view with derived location context.

## Task

Create or replace a view named vw_low_stock_inventory that returns warehouse_id, warehouse_code, product_id, sku, quantity_on_hand, and reorder_point for inventory records whose quantity_on_hand is at or below reorder_point. Then return every row from vw_low_stock_inventory.

## Result requirements

- **Ordering:** Order by warehouse_id ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write a PostgreSQL `CREATE OR REPLACE VIEW ... AS` statement followed by one `SELECT` statement in `exercises/27-views/q134.sql`. The final SELECT must return the requested view result. The grader runs both statements in a transaction and rolls it back.
