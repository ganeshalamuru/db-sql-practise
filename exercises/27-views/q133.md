# Q133 - VIEWS

- **Difficulty:** 3 / 5
- **Learning objective:** Create a grouped analytical view and expose its aggregate result columns.

## Task

Create or replace a view named vw_product_review_summary that returns product_id, sku, review_count, and average_rating for products with at least one review. average_rating is the average review rating. Then return every row from vw_product_review_summary.

## Result requirements

- **Ordering:** Order by review_count descending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write a PostgreSQL `CREATE OR REPLACE VIEW ... AS` statement followed by one `SELECT` statement in `exercises/27-views/q133.sql`. The final SELECT must return the requested view result. The grader runs both statements in a transaction and rolls it back.
