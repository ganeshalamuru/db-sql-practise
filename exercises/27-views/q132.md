# Q132 - VIEWS

- **Difficulty:** 2 / 5
- **Learning objective:** Encapsulate an aggregate over related rows in a reusable view.

## Task

Create or replace a view named vw_delivered_order_totals that returns order_id, customer_id, ordered_at, and item_total for delivered orders. item_total is the sum of quantity multiplied by unit_price across each order's items. Then return every row from vw_delivered_order_totals.

## Result requirements

- **Ordering:** Order by ordered_at ascending, then order_id ascending.
- **Duplicates:** Duplicates are significant.

Write a PostgreSQL `CREATE OR REPLACE VIEW ... AS` statement followed by one `SELECT` statement in `exercises/27-views/q132.sql`. The final SELECT must return the requested view result. The grader runs both statements in a transaction and rolls it back.
