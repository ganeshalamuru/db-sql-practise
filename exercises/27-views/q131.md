# Q131 - VIEWS

- **Difficulty:** 1 / 5
- **Learning objective:** Create a simple filtered and projected view, then query it.

## Task

Create or replace a view named vw_active_products that returns product_id, sku, name, and unit_price for active products. Then return every row from vw_active_products.

## Result requirements

- **Ordering:** Order by product_id ascending.
- **Duplicates:** Duplicates are significant.

Write a PostgreSQL `CREATE OR REPLACE VIEW ... AS` statement followed by one `SELECT` statement in `exercises/27-views/q131.sql`. The final SELECT must return the requested view result. The grader runs both statements in a transaction and rolls it back.
