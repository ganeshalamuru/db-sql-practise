# Q020 - Expressions

- **Difficulty:** 3 / 5
- **Learning objective:** Use a scalar expression to convert a monetary amount to another unit.

## Task

Return product_id, sku, unit_price, and price_in_cents, where price_in_cents is unit_price multiplied by 100.

## Result requirements

- **Ordering:** Order by product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/04-expressions/q020.sql`. Do not modify the schema or data.
