# Q095 - RECURSIVE CTE

- **Difficulty:** 5 / 5
- **Learning objective:** Combine recursive traversal, joins, and aggregation to summarize a hierarchy.

## Task

Return root_category_id, root_category_name, category_id, category_name, hierarchy_level, and active_product_count for every category in each root category's hierarchy. active_product_count is the number of active products assigned directly to that category, including zero counts.

## Result requirements

- **Ordering:** Order by root_category_id ascending, hierarchy_level ascending, then category_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/19-recursive-cte/q095.sql`. Do not modify the schema or data.
