# Q057 - CROSS JOIN

- **Difficulty:** 4 / 5
- **Learning objective:** Create combinations first, then exclude pairs that do not meet a comparison rule.

## Task

Return store_id, store_code, warehouse_id, and warehouse_code for every store-and-warehouse pair located in different cities.

## Result requirements

- **Ordering:** Order by store_id ascending, then warehouse_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/12-cross-join/q057.sql`. Do not modify the schema or data.
