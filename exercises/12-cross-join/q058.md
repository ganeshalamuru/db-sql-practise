# Q058 - CROSS JOIN

- **Difficulty:** 4 / 5
- **Learning objective:** Create a planning matrix between two business dimensions.

## Task

Return supplier_id, supplier_name, category_id, and category_name for every possible supplier-and-category combination.

## Result requirements

- **Ordering:** Order by supplier_id ascending, then category_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/12-cross-join/q058.sql`. Do not modify the schema or data.
