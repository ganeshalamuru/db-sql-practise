# Q115 - DATE FUNCTIONS

- **Difficulty:** 5 / 5
- **Learning objective:** Use LAG and date arithmetic to find closely spaced customer orders.

## Task

Return customer_id, order_id, ordered_date, previous_order_id, previous_ordered_date, and days_since_previous_order for orders placed no more than 120 days after the previous order by the same customer. Determine the previous order chronologically by ordered_at and then order_id. ordered_date and previous_ordered_date are calendar dates.

## Result requirements

- **Ordering:** Order by customer_id ascending, ordered_date ascending, then order_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/23-date-functions/q115.sql`. Do not modify the schema or data.
