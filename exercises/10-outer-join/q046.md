# Q046 - OUTER JOIN

- **Difficulty:** 3 / 5
- **Learning objective:** Use a LEFT JOIN to find rows with no related match.

## Task

Return customer_id, email, first_name, and last_name for customers who have never placed an order.

## Result requirements

- **Ordering:** Order by customer_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/10-outer-join/q046.sql`. Do not modify the schema or data.
