# Q047 - OUTER JOIN

- **Difficulty:** 4 / 5
- **Learning objective:** Count optional related rows while retaining rows with no matches.

## Task

Return product_id, product_name, and review_count for every active product, including products with no reviews.

## Result requirements

- **Ordering:** Order by review_count descending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/10-outer-join/q047.sql`. Do not modify the schema or data.
