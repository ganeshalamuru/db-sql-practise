# Q049 - OUTER JOIN

- **Difficulty:** 5 / 5
- **Learning objective:** Use COALESCE with an outer join to return zero instead of NULL aggregate results.

## Task

Return warehouse_id, warehouse_code, and total_units_on_hand for every warehouse, including warehouses with no inventory rows. Show 0 rather than NULL for total_units_on_hand.

## Result requirements

- **Ordering:** Order by total_units_on_hand descending, then warehouse_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/10-outer-join/q049.sql`. Do not modify the schema or data.
