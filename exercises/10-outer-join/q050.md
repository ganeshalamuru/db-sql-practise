# Q050 - OUTER JOIN

- **Difficulty:** 5 / 5
- **Learning objective:** Count conditionally matched rows without removing unmatched parent rows.

## Task

Return product_id, product_name, and open_return_count for every active product. Count only returns whose status is requested or approved, and include products with no such returns.

## Result requirements

- **Ordering:** Order by open_return_count descending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/10-outer-join/q050.sql`. Do not modify the schema or data.
