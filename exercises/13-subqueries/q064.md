# Q064 - SUBQUERIES

- **Difficulty:** 5 / 5
- **Learning objective:** Use a correlated aggregate subquery to identify the top row within each group.

## Task

Return warehouse_id, warehouse_code, product_id, sku, quantity_on_hand, and reorder_point for inventory rows whose quantity_on_hand is the highest in their warehouse. Include every tied highest row.

## Result requirements

- **Ordering:** Order by warehouse_id ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/13-subqueries/q064.sql`. Do not modify the schema or data.
