# Q087 - CTE

- **Difficulty:** 3 / 5
- **Learning objective:** Use a CTE to calculate reusable line-item totals.

## Task

Return order_id, ordered_at, status, and item_total for orders with an item_total of at least 1,000.00. item_total is the sum of quantity multiplied by unit_price across each order's line items.

## Result requirements

- **Ordering:** Order by item_total descending, then order_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/18-cte/q087.sql`. Do not modify the schema or data.
