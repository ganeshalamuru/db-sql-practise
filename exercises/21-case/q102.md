# Q102 - CASE

- **Difficulty:** 5 / 5
- **Learning objective:** Classify inventory health from multiple aggregate conditions.

## Task

Return product_id, sku, warehouse_count, low_stock_warehouse_count, and inventory_health for every active product. warehouse_count is the number of warehouses that stock the product. low_stock_warehouse_count is the number of those rows where quantity_on_hand is at or below reorder_point. inventory_health is not_stocked when warehouse_count is 0, reorder_needed when low_stock_warehouse_count is positive, and adequately_stocked otherwise.

## Result requirements

- **Ordering:** Order by inventory_health ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/21-case/q102.sql`. Do not modify the schema or data.
