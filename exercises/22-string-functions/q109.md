# Q109 - STRING FUNCTIONS

- **Difficulty:** 5 / 5
- **Learning objective:** Produce deterministic ordered text summaries from related review records.

## Task

Return product_id, sku, review_count, reviewer_email_list, and review_text_summary for products with at least two reviews. reviewer_email_list contains reviewer emails separated by ', ' and sorted by email ascending. review_text_summary contains each review body, using '(no text)' for NULL bodies, separated by ' | ' and sorted by created_at ascending and then review_id ascending.

## Result requirements

- **Ordering:** Order by review_count descending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/22-string-functions/q109.sql`. Do not modify the schema or data.
