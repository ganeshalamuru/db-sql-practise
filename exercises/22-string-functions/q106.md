# Q106 - STRING FUNCTIONS

- **Difficulty:** 5 / 5
- **Learning objective:** Build hierarchical display text while traversing a recursive structure.

## Task

Return category_id, category_name, parent_category_id, hierarchy_level, and category_path for every category. hierarchy_level is 0 for root categories. category_path contains names from the root to the category separated by ' > '. Use string construction as part of the recursive traversal.

## Result requirements

- **Ordering:** Order by category_path ascending, then category_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/22-string-functions/q106.sql`. Do not modify the schema or data.
