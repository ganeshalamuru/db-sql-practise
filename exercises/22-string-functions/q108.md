# Q108 - STRING FUNCTIONS

- **Difficulty:** 5 / 5
- **Learning objective:** Use ordered string aggregation after multi-table grouping and filtering.

## Task

Return supplier_id, supplier_name, category_id, category_name, active_product_count, and active_sku_list for each supplier-and-category combination with at least two active products. active_sku_list contains that combination's active SKUs separated by ', ' and sorted by SKU ascending.

## Result requirements

- **Ordering:** Order by supplier_id ascending, then category_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/22-string-functions/q108.sql`. Do not modify the schema or data.
