# Q143 - UPDATE

- **Difficulty:** 3 / 5
- **Learning objective:** Use UPDATE ... FROM to update rows selected through a related relation.

## Task

For inventory records of inactive products, set reorder_point to quantity_on_hand plus 10. Return warehouse_id, product_id, quantity_on_hand, and reorder_point for each updated inventory record.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `UPDATE` statement in `exercises/29-update/q143.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
