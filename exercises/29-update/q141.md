# Q141 - UPDATE

- **Difficulty:** 1 / 5
- **Learning objective:** Update one row with a direct predicate and inspect the new values.

## Task

Update customer_id 1 so its status is 'suspended'. Return customer_id, email, and status for the updated customer.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `UPDATE` statement in `exercises/29-update/q141.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
