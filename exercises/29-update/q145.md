# Q145 - UPDATE

- **Difficulty:** 5 / 5
- **Learning objective:** Use a CTE to identify a multi-row update population from an aggregate condition.

## Task

For every inventory record belonging to a product that has at least one warehouse where quantity_on_hand is at or below reorder_point, increase reorder_point by 5. Return warehouse_id, product_id, quantity_on_hand, and reorder_point for each updated inventory record.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `UPDATE` statement in `exercises/29-update/q145.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
