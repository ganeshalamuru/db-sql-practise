# Q142 - UPDATE

- **Difficulty:** 2 / 5
- **Learning objective:** Update a filtered group of rows with an arithmetic expression.

## Task

Reduce unit_price by 10 percent for products supplied by supplier_id 40. Return product_id, sku, and unit_price for each updated product.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `UPDATE` statement in `exercises/29-update/q142.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
