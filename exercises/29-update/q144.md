# Q144 - UPDATE

- **Difficulty:** 4 / 5
- **Learning objective:** Use a correlated EXISTS predicate to update records with qualifying related rows.

## Task

Update reviews with a NULL body for active products, setting body to 'Review body pending moderation'. Return review_id, product_id, customer_id, and body for each updated review.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `UPDATE` statement in `exercises/29-update/q144.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
