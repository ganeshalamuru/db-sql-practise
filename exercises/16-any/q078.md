# Q078 - ANY

- **Difficulty:** 4 / 5
- **Learning objective:** Use a correlated ANY subquery to compare values within a group.

## Task

Return product_id, product_name, category_id, and unit_price for active products that cost more than at least one other active product in the same category.

## Result requirements

- **Ordering:** Order by category_id ascending, unit_price descending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/16-any/q078.sql`. Do not modify the schema or data.
