# Q080 - ANY

- **Difficulty:** 5 / 5
- **Learning objective:** Use ANY in a grouped query with a related-table filter.

## Task

Return supplier_id, supplier_name, and active_product_count for suppliers with at least two active products, where at least one active product costs more than any active product in category_id 1.

## Result requirements

- **Ordering:** Order by active_product_count descending, then supplier_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/16-any/q080.sql`. Do not modify the schema or data.
