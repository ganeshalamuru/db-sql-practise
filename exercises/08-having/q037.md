# Q037 - HAVING

- **Difficulty:** 3 / 5
- **Learning objective:** Use HAVING to filter grouped sums.

## Task

Return warehouse_id and total_units_on_hand for warehouses whose total quantity_on_hand is greater than 50,000.

## Result requirements

- **Ordering:** Order by total_units_on_hand descending, then warehouse_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/08-having/q037.sql`. Do not modify the schema or data.
