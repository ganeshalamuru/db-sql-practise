# Q010 - WHERE

- **Difficulty:** 2 / 5
- **Learning objective:** Compare one column to another in a row predicate.

## Task

Return warehouse_id, product_id, quantity_on_hand, and reorder_point for inventory rows at or below their reorder point.

## Result requirements

- **Ordering:** Order by warehouse_id ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/02-where/q010.sql`. Do not modify the schema or data.
