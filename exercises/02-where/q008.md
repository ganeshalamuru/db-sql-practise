# Q008 - WHERE

- **Difficulty:** 2 / 5
- **Learning objective:** Combine predicates with AND.

## Task

Return order_id, customer_id, ordered_at, and status for delivered orders placed during 2024.

## Result requirements

- **Ordering:** Order by ordered_at ascending, then order_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/02-where/q008.sql`. Do not modify the schema or data.
