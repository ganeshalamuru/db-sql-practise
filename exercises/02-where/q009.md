# Q009 - WHERE

- **Difficulty:** 2 / 5
- **Learning objective:** Filter values with IN.

## Task

Return payment_id, order_id, payment_method, amount, and status for completed payments made by card or wallet.

## Result requirements

- **Ordering:** Order by payment_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/02-where/q009.sql`. Do not modify the schema or data.
