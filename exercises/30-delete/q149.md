# Q149 - DELETE

- **Difficulty:** 4 / 5
- **Learning objective:** Use a correlated EXISTS predicate to remove records with a qualifying related row.

## Task

Delete returns whose order has a payment with status failed. Return return_id, order_id, product_id, and status for each deleted return.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `DELETE` statement in `exercises/30-delete/q149.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
