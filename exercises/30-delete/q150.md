# Q150 - DELETE

- **Difficulty:** 5 / 5
- **Learning objective:** Use a CTE to derive multi-table deletion criteria before deleting and returning the affected records.

## Task

Delete warehouse inventory records for inactive products whose quantity_on_hand is at or above reorder_point in every warehouse where they are stocked. Return warehouse_id, product_id, quantity_on_hand, and reorder_point for each deleted inventory record.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `DELETE` statement in `exercises/30-delete/q150.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
