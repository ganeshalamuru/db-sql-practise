# Q147 - DELETE

- **Difficulty:** 2 / 5
- **Learning objective:** Use DELETE ... USING to target rows through a related table.

## Task

Delete store inventory records for inactive products. Return store_id, product_id, and quantity_on_hand for each deleted inventory record.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `DELETE` statement in `exercises/30-delete/q147.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
