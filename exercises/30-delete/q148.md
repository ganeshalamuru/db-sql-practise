# Q148 - DELETE

- **Difficulty:** 3 / 5
- **Learning objective:** Use a subquery to delete rows selected by a relationship condition.

## Task

Delete reviews written by customers whose status is closed. Return review_id, product_id, and customer_id for each deleted review.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `DELETE` statement in `exercises/30-delete/q148.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
