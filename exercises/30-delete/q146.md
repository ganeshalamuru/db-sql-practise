# Q146 - DELETE

- **Difficulty:** 1 / 5
- **Learning objective:** Delete rows with a direct predicate and inspect the affected keys.

## Task

Delete every review whose body is NULL. Return review_id, product_id, and customer_id for each deleted review.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `DELETE` statement in `exercises/30-delete/q146.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
