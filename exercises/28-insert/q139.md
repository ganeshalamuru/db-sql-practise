# Q139 - INSERT

- **Difficulty:** 4 / 5
- **Learning objective:** Populate multiple rows with INSERT ... SELECT from a derived source set.

## Task

Insert inventory records for warehouse_id 10 and product IDs 1 through 5. Each inserted row must have quantity_on_hand 0 and reorder_point 25. Return warehouse_id, product_id, quantity_on_hand, and reorder_point for each inserted inventory record.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `INSERT` statement in `exercises/28-insert/q139.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
