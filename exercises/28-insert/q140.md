# Q140 - INSERT

- **Difficulty:** 5 / 5
- **Learning objective:** Insert a derived set only where a related row does not already exist.

## Task

Insert store inventory records for product_id 11 at every store that does not already stock that product. Each inserted record must have quantity_on_hand 25. Return store_id, product_id, and quantity_on_hand for each inserted store inventory record.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `INSERT` statement in `exercises/28-insert/q140.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
