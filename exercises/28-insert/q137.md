# Q137 - INSERT

- **Difficulty:** 2 / 5
- **Learning objective:** Insert a row that satisfies foreign-key and typed-column requirements.

## Task

Insert an active product with supplier_id 1, category_id 1, sku 'SKU-PRACTICE-INSERT-137', name 'Practice Insert Product', unit_price 49.99, and created_at '2024-01-15 09:00:00+00'. Return sku, name, unit_price, is_active, and created_at for the inserted product.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `INSERT` statement in `exercises/28-insert/q137.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
