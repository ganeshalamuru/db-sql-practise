# Q138 - INSERT

- **Difficulty:** 3 / 5
- **Learning objective:** Insert a related record while respecting a composite uniqueness rule.

## Task

Insert a review for product_id 100 and customer_id 1 with rating 5, body 'Excellent practice product', and created_at '2024-02-01 12:30:00+00'. Return product_id, customer_id, rating, body, and created_at for the inserted review.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `INSERT` statement in `exercises/28-insert/q138.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
