# Q136 - INSERT

- **Difficulty:** 1 / 5
- **Learning objective:** Insert a single row with explicit column values and inspect the inserted data.

## Task

Insert a supplier with name 'Practice Insert Supplier', contact_email 'practice-insert-supplier@example.test', and country_id 1. Return name, contact_email, and country_id for the inserted supplier.

## Result requirements

- **Ordering:** No ordering is required.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `INSERT` statement in `exercises/28-insert/q136.sql`. Include a `RETURNING` clause with the requested columns. The grader runs the statement in a transaction and rolls it back.
