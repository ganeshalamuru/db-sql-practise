# Q072 - IN

- **Difficulty:** 3 / 5
- **Learning objective:** Use NOT IN with a non-NULL subquery result to exclude matched rows.

## Task

Return customer_id, email, first_name, and last_name for customers who have not placed an order with status paid, shipped, or delivered.

## Result requirements

- **Ordering:** Order by customer_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/15-in/q072.sql`. Do not modify the schema or data.
