# Q014 - ORDER BY

- **Difficulty:** 2 / 5
- **Learning objective:** Order by a computed expression.

## Task

Return order_id, product_id, quantity, unit_price, and line_total (quantity multiplied by unit_price) for all order items.

## Result requirements

- **Ordering:** Order by line_total descending, then order_id ascending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/03-order-by/q014.sql`. Do not modify the schema or data.
