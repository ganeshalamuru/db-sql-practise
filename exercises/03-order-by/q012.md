# Q012 - ORDER BY

- **Difficulty:** 1 / 5
- **Learning objective:** Sort text values alphabetically with a deterministic tie-breaker.

## Task

Return supplier_id, name, and contact_email for all suppliers.

## Result requirements

- **Ordering:** Order by name ascending, then supplier_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/03-order-by/q012.sql`. Do not modify the schema or data.
