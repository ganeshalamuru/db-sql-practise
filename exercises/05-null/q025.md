# Q025 - NULL

- **Difficulty:** 3 / 5
- **Learning objective:** Use COALESCE to produce a display-ready text value.

## Task

Return review_id, product_id, customer_id, rating, and review_text. Replace a NULL review body with the text No written review.

## Result requirements

- **Ordering:** Order by review_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/05-null/q025.sql`. Do not modify the schema or data.
