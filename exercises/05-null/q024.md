# Q024 - NULL

- **Difficulty:** 2 / 5
- **Learning objective:** Use a NULL condition together with another predicate.

## Task

Return order_id, customer_id, ordered_at, and status for cancelled orders that have no promotion applied.

## Result requirements

- **Ordering:** Order by order_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/05-null/q024.sql`. Do not modify the schema or data.
