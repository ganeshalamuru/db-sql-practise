# Q022 - NULL

- **Difficulty:** 1 / 5
- **Learning objective:** Find present values with IS NOT NULL.

## Task

Return shipment_id, order_id, shipped_at, and delivered_at for shipments that have been delivered.

## Result requirements

- **Ordering:** Order by shipment_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/05-null/q022.sql`. Do not modify the schema or data.
