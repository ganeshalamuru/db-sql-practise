# Q045 - INNER JOIN

- **Difficulty:** 5 / 5
- **Learning objective:** Traverse several foreign-key relationships in one query.

## Task

Return shipment_id, shipment_status, warehouse_code, warehouse_name, city_name, and country_name for delivered shipments.

## Result requirements

- **Ordering:** Order by country_name ascending, city_name ascending, warehouse_code ascending, then shipment_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/09-inner-join/q045.sql`. Do not modify the schema or data.
