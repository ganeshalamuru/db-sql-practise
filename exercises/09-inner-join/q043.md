# Q043 - INNER JOIN

- **Difficulty:** 4 / 5
- **Learning objective:** Join order headers to line items and calculate a derived value.

## Task

Return order_id, ordered_at, product_id, quantity, unit_price, and line_total for order items on paid, shipped, or delivered orders. line_total is quantity multiplied by unit_price.

## Result requirements

- **Ordering:** Order by ordered_at descending, then order_id descending, then product_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/09-inner-join/q043.sql`. Do not modify the schema or data.
