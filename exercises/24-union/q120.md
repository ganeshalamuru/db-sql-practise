# Q120 - UNION

- **Difficulty:** 5 / 5
- **Learning objective:** Aggregate a UNION ALL event stream into customer-level activity metrics.

## Task

Return customer_id, email, order_count, review_count, and last_activity_at for every customer with at least one order or review. order_count counts orders, review_count counts reviews, and last_activity_at is the latest ordered_at or review created_at timestamp for that customer.

## Result requirements

- **Ordering:** Order by last_activity_at descending, then customer_id ascending.
- **Duplicates:** Duplicates are significant.

Write one PostgreSQL `SELECT` statement in `exercises/24-union/q120.sql`. Do not modify the schema or data.
